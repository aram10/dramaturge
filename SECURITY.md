# Security Policy

## Supported Versions

The project is pre-1.0 and only the latest released version (and `main`) receives security fixes. Please upgrade to the most recent release before requesting a fix.

## Reporting a Vulnerability

Please report security issues privately using one of these channels:

- [GitHub Security Advisories](https://github.com/aram10/dramaturge/security/advisories/new) (preferred)
- Email: security@dramaturge.dev

Do not open public issues for suspected vulnerabilities.

We aim to acknowledge new reports within 3 business days and provide a resolution or mitigation timeline after initial triage. If you have not received a response in that window, please follow up using the same channel.
